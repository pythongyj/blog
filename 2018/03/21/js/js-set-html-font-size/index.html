1.window.getComputedStyle(element, [pseudoElt]) 获取指定元素对应CSS属性的最终计算值

  • 官方解释

  • 参数解析

    • element：必需，要获取样式值的元素对象
    • pseudoElt：可选，表示指定节点的伪元素（:before、:after、:first-line、:first-letter等）

  • 兼容性

    • IE9+浏览器支持此方法

    • 谷歌浏览器支持此方法
